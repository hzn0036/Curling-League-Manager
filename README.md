# Curling League Manager

This is my final project for CPSC 4973. It is a desktop application with a GUI built using PySide6 that lets you manage curling leagues, teams, and team members.

## Requirements
- Python 3.14
- PySide6
- yagmail

## Installation
Install the required packages by running:
pip install -r requirements.txt

## How to Run
Open a terminal in the project folder and run:
python curling_league/main_window.py

## What it does
- The main window shows all leagues in the database
- You can add and delete leagues
- You can edit a league to manage its teams
- In the league editor you can add, delete, and edit teams
- You can import and export team data using CSV files
- You can edit a team to manage its members
- You can add, update, and delete team members
- The File menu lets you load and save the database
